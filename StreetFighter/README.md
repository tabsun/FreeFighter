StreetFighter 街头霸王
======================
'90年代的感觉

By [AlloyTeam](http://www.AlloyTeam.com/)

### 主机:
  
   * 移动: W: 上, D: 前, A: 后, S：下 
   * 攻击: J: 轻拳, K: 重拳, U: 轻腿, I: 重腿
   * 技能: 下→前→拳: 波动拳, 下→后→腿：旋风腿, 前→下→前→拳：升龙拳
   
### 副机（小键盘）:
  
   * 移动: ↑: 上, ←: 前, →: 后, ↓：下 
   * 攻击: 1: 轻拳, 2: 重拳, 4: 轻腿, 5: 重腿
   * 技能: 下→前→拳: 波动拳, 下→后→腿：旋风腿, 前→下→前→拳：升龙拳
   
   

按F2暂停游戏, 1键大战电脑ai, 2键双人对打.
